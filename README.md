Para ligar o Front
- acessar a pasta frontend
- acessar o terminal
- npm Install
- npm run dev
- Acessar o locao indicado no terminal pelo seu navegador
-- É importante ter o node instalado

Para ligar o flask
- acessar a pasta flask dentro da pasta backend
- acessar o terminal
- digitar no terminal: python app.py
-- É importante ter Python e flask instalados

Para ligar o worker
- acessar a pasta worker dentro da pasta backend
- acessar o terminal
- digitar no terminal: python worker.py
-- É importante ter Python o sistema de filas já esta hospedado na nuvem para maior praticidade
